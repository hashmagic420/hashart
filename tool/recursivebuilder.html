GUIDES


3D Recursive HTML:

1. Create a 3D object (glb format) using any program, for example Blender
2. Inscribe glb file in Doge blockchain
3. Use it to create a recursion by simply pasting inscription ID into the model-viewer section.
4. Save the HTML file
5. Inscribe HTML file in Doge blockchain

IMAGE Recursive HTML:

1. Create images (jpg, webp, png format) using any program
2. Inscribe in Doge blockchain
3. Use it to create a recursion by simply pasting inscription ID into the img section.
4. Save the HTML file
5. Inscribe HTML file in Doge blockchain

HTML:

1. Write any code without using third-party scripts, as in the example
2. Save the HTML file
3. Inscribe HTML file in Doge blockchain

Disclaimer! Don't use the example as a finished recursive product! You need to create an object yourself (be it 3D, image or HTML code), inscribe it to Dogecoin blockchain, create a recursion (using the example in the builder) and inscribe it to the blockchain as an HTML file.
